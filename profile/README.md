# Gonggoo 프로젝트
공동구매할 학생을 이어주는 프로젝트입니다.

## 프로젝트 개발 목적
온라인으로 물품을 구매하게 되면 박스 단위로 구매해야 하는 경우가 많습니다.

혼자서 대량으로 물품을 구매하기는 어려울 때 
물품의 공동구매를 쉽게 하는것이 이 프로젝트의 목적입니다.
